
# Desafio B2BFlow - Integração Z-API e Supabase

Projeto desenvolvido para a 1ª etapa do processo seletivo de Estágio em Desenvolvimento Python da b2bflow. O objetivo deste script é ler contatos de um banco de dados no Supabase e disparar mensagens personalizadas via WhatsApp utilizando a Z-API.

##  Setup da Tabela (Supabase)

Para o script funcionar, é necessário criar a tabela `contatos` no seu projeto do Supabase. Utilize o código SQL abaixo no *SQL Editor*:

```sql
CREATE TABLE contatos (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome_contato text NOT NULL,
  telefone text NOT NULL
);
