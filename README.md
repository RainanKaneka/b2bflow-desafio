
# Desafio B2BFlow - Integração Z-API e Supabase

Projeto desenvolvido para a 1ª etapa do processo seletivo de Estágio em Desenvolvimento Python da b2bflow. O objetivo deste script é ler contatos de um banco de dados no Supabase e disparar mensagens personalizadas via WhatsApp utilizando a Z-API.

##  Setup da Tabela (Supabase)


Para o script funcionar, é necessário criar a tabela `contatos` no seu projeto do Supabase e inserir dados de teste. Utilize o código SQL abaixo no *SQL Editor*:

```sql
CREATE TABLE contatos (
  id bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome_contato text NOT NULL,
  telefone text NOT NULL
); 
```
-- ATENÇÃO: Substitua o número abaixo pelo número que você quer enviar a mensagem.

```sql
INSERT INTO contatos (nome_contato, telefone) 
VALUES ('nome', '5511999999999');
```


##  Como Rodar o Projeto

1. Clone o repositório.
2. Crie um ambiente virtual: `python -m venv venv`
3. Ative o ambiente virtual:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. Instale as dependências: `pip install -r requirements.txt`
5. Configure o arquivo `.env` com suas credenciais.
6. Execute o script principal: `python main.py`
